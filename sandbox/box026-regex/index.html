😬 RegEx 😬

Default HTML "pattern=" matching:

[abcd]
Matches a single instance of one of the characters in the brackets. Can only match one character at a time.
abcd
Matches only the exact sequence entered, "abcd".
ab*c
Matches a single "a", followed by any amount of "b", followed by a single "c".

Scripted inputs:

^A
Matches so long as the string begins with "A".
^abcd*e
Matches if the string begins with "abc" followed by none or any number of "d" and one "e".
p[el]
Matches if there is a "p" followed by an "e" or an "l" anywhere in the text. Order of ocurrence matters.
ed&
Matches if there the string ends in "ed".
\w
Matches in the presence of any word character. Space is a non-word character so 
     m
and 
*******m
match despite the leading whitespace and asterisks.
oon\b
Matches if the pattern ("oon") has a non-word character boundary at the end like " " or "*"
\Boo
Matches if there is a word character at the beginning of the string only.
\Boo\b (heh)
Matches if there is a word character at the beginning of the string and a non-word character at the end of the string.
