😬 RegEx 😬

Scripted inputs:

\d
Matches if the string contains a number character.
\d{5}
Matches matches if there are five consecutive number characters anywhere in string. Doesn't count if there are other characters in between the numbers.
(?<!\w)\d{5}(?>!\w)
Matches if there is a non-number character anywhere in the string.
\D
Matches if there is a non-number character anywhere in the string.
\w
Matches any alphanumeric character. Characters like "-" and "@" will not match. "_" Matches because of the Perl thing.
\W
Matches only non-alphanumeric character. Will not match "_" due to the Perl thing.
\s
Matches the whitespace character inside any string. Includes space, tab, form feed (?), line feed (?), and "other unicode spaces".
\S
Matches the opposite of the above. Does not match if there is a non-whitespace character anywhere in the string.
apple|orange
Disjunction / "or" operation: Matches if the characters on either side of the bar are present.
^apple$|^orange$
Matched the full string "apple" or "orange", so long ad there is nothing before or after the string.
