😬 RegEx 😬

Scripted inputs:

x(?=y)
Matches if "x" is follow by "y" at any point within the string.
fizz(?buzz|10)
Matches if fizz is "follow" by "buzz" or "10" anywhere in the string.
\d
Matches a number anywhere in the string.
x(?!y)
Matches matches "x" if "x" is not followed by a y. Followed includes non-word characters.
(?<=y)x
Matches matches only if "x" is preceded by a "y" anywhere in the string.
(?<!y)x
Matches matches only if "x" is NOT preceded by a "y" anywhere in the string.

Character classes:

square brackets: [abcdefg]
[a-g]
[-abcdef]
Matches any of the characters enclosed in the square brackets.
Matches the range indicated by the hyphen.
If the hyphen occurs as the first or last character it is included as a character to be matched.
square brackets: [^abcdefg]
Matches anything that is not the characters included in the square brackets. Same hyphen rules as above.
